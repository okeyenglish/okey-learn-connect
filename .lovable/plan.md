
# План: Нормализация размера элементов списка преподавателей в мобильной версии

## Проблема

На скриншоте видно, что элементы списка чатов преподавателей имеют немного увеличенную высоту по сравнению со стандартными клиентскими чатами. Это вызвано:

1. **Разная высота виртуализации**: `ITEM_HEIGHT = 68` в преподавателях vs `estimateSize: 60` в клиентах
2. **Дополнительные бейджи категорий** ("Школьники", "Дошкольники") занимают отдельную строку
3. **Отступы между элементами** (`mb-0.5`) накапливаются

## Решение

### Изменение 1: TeacherChatList.tsx

Уменьшить высоту элементов виртуализации для соответствия клиентскому списку:

```typescript
// Было:
const ITEM_HEIGHT = 68;
const GROUP_CHAT_HEIGHT = 68;

// Станет:
const ITEM_HEIGHT = 60;
const GROUP_CHAT_HEIGHT = 60;
```

### Изменение 2: TeacherListItem.tsx

Оптимизировать компактность элемента списка:

1. **Уменьшить размер аватара на мобильных**: `h-8 w-8` вместо `h-9 w-9`
2. **Сделать бейджи категорий inline** с именем вместо отдельной строки
3. **Уменьшить padding**: `p-1.5` вместо `p-2` на мобильных
4. **Убрать лишний отступ** `mb-0.5` у строки с бейджами

Финальная структура элемента:
```text
[Аватар] [Имя + флаги + пин] ............... [Время]
         [Бейджи категорий (inline, меньше)]  [Счётчик]
         [Текст последнего сообщения]
```

### Изменение 3: TeacherChatList.tsx (группа педагогов)

Синхронизировать стили группового чата с TeacherListItem:
- Уменьшить аватар до `h-8 w-8`
- Уменьшить padding до `p-1.5`

## Технические детали

### TeacherListItem.tsx - обновлённые классы

```tsx
// Кнопка - уменьшенный padding
className={`w-full p-1.5 md:p-2 text-left rounded-lg ...`}

// Аватар - меньше на мобильных
<Avatar className="h-8 w-8 md:h-9 md:w-9 flex-shrink-0 ...">

// Бейджи категорий - более компактные
className="text-[8px] h-3 px-1 py-0 ..."

// Имя и бейджи в одной строке
<div className="flex items-center gap-1 flex-wrap">
  <span className="text-sm truncate ...">...</span>
  {categoryBadges inline here}
</div>
```

### TeacherChatList.tsx - групповой чат

```tsx
// Аватар группы
<div className="h-8 w-8 md:h-9 md:w-9 flex-shrink-0 ...">

// Padding
className="w-full p-1.5 md:p-2 text-left rounded-lg ..."
```

## Файлы для изменения

## Результат

После изменений элементы списка преподавателей будут иметь такую же высоту и пропорции, как элементы списка клиентов (~60px), обеспечивая единообразный UX в мобильной версии CRM.
